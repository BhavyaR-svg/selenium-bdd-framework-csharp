#  Selenium BDD Automation Framework (C#)

##  Overview
This is a robust Selenium automation framework built using **C#, NUnit, and Reqnroll (BDD)**.  
It follows best practices like **Page Object Model, parallel execution, logging, and reporting**.

---

## Tech Stack
- C#
- Selenium WebDriver
- NUnit
- Reqnroll (BDD)
- Extent Reports
- Serilog (Logging)

---

## Key Features
- BDD Framework using Reqnroll
- Page Object Model (POM)
- Extent HTML Reporting
- Serilog Logging (file + console)
- JSON Data-driven Testing
- Screenshot capture on failure
- Clean and scalable architecture
